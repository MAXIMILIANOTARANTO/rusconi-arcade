# 🎮 RUSCONI Arcade 80s·90s

**Versión Final - Lista para Humanos**

Experiencia arcade retro interactiva con estética de los 80s y 90s, mecánicas de juego reales y un emotivo overlay de Campeón del Mundo.

## ✨ Características

- Sistema de puntuación funcional + high score persistente
- Botón principal de interacción (HIT THE BUNNY)
- Video con loop inteligente
- Pantalla de Campeón con carga de foto e himno
- Confeti y efectos visuales
- Totalmente optimizado para móvil
- **PWA instalable** como aplicación

## Cómo jugar

1. Tocá **TOCA PARA EMPEZAR**
2. Usá el botón rosa **HIT THE BUNNY** para sumar puntos
3. Intentá superar tu récord
4. Tocá **VER CAMPEÓN** para abrir la pantalla emotiva
5. Cargá tu foto del 18 de diciembre y el himno

## Cómo instalarla como App (PWA)

1. Abrí el link en Chrome o Safari del celular
2. Tocá los tres puntitos → **Agregar a pantalla de inicio**
3. Listo. Se instala como aplicación nativa.

## Link de la experiencia

**https://maximilianotaranto.github.io/rusconi-arcade/** (después de activar GitHub Pages)

## Cómo activar GitHub Pages (4 pasos)

1. Entrá al repositorio: https://github.com/MAXIMILIANOTARANTO/rusconi-arcade
2. Andá a **Settings → Pages**
3. En **Source** seleccioná:
   - Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
4. Tocá **Save**

En menos de 1 minuto tendrás el link limpio funcionando.
